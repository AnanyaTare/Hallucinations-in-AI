# Deployment & Demos

This directory contains configurations, API scripts, and demo apps (e.g., Streamlit, Gradio) to showcase the hallucination detection and mitigation framework in real-time.

## 📂  Structure
```text
deployment/
├── app/            # Interactive web app code (e.g., Streamlit dashboard)
├── api/            # API wrappers (FastAPI/Flask) for the evaluation engine
└── docker/         # Dockerfiles and docker-compose files for local deployment
```
